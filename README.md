# 🧪 物理实验资料库

> 面向大学物理课程的中文物理教育资源库，包含丰富的实验文档、评估材料和教学资源。

## 📁 目录结构

本仓库采用三层结构进行组织：

- **📚 `document/`** - 存放正式的实验指南、教材和理论背景知识
- **📝 `paper/`** - 包含评估材料、带有详细解答和说明的考题
- **🎯 `study_guide/`** - 提供针对各项实验的学习指南
- **📂 `原文件/`** - 作为参考的原始补充文件（PDF、PPT、DOC等）

## 🔬 主要实验内容

| 实验名称 | 描述 | 理论基础 |
|---------|------|----------|
| **🔊 声速与光速测量** | 测量声波和光波在介质中的传播速度 | 波动理论、多普勒效应 |
| **🔆 太阳能电池特性** | 研究光伏效应和太阳能电池的工作特性 | 半导体物理、光电效应 |
| **🧲 巨磁电阻** | 探索磁性材料的电阻变化现象 | 磁学、凝聚态物理 |
| **🏥 核磁共振** | 利用核磁共振现象研究物质结构 | 量子力学、磁共振 |
| **⚪ 落球法测粘滞系数** | 通过小球下落测量液体粘滞系数 | 流体力学、斯托克斯定律 |
| **🌈 迈克尔逊干涉仪** | 观察光的干涉现象并测量光波长 | 光学、波动光学 |

## 🚀 快速开始

1. **选择实验**: 从上方实验表格中选择要学习的实验
2. **理论学习**: 先阅读 `document/` 中的理论讲义
3. **实验操作**: 按照实验指导进行实践操作
4. **练习评估**: 完成 `paper/` 中的考题检验学习效果
5. **复习巩固**: 参考 `study_guide/` 中的学习建议

## 📈 学习建议

- 🔍 **理论联系实际**: 理解实验原理后再进行操作
- 📊 **重视数据处理**: 学会分析实验数据和误差
- 🤔 **思考物理意义**: 不仅是操作，更要理解背后的物理原理
- 📝 **做好实验记录**: 详细记录实验过程和结果
